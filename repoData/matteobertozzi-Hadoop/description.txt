Hadoop Patches
------------------------------------------------------------------------------
HFILE-remove-double-write.patch - Remove double write, if cache, and speedup write.
HFILE-appendMetaBlock.patch - Replace sequential search with binary search.


Pure Python SequenceFile Reader and Writer implementation
that allows you to read and write your Hadoop sequence files
without using java.
