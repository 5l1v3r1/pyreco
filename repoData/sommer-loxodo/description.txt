Quickstart:
-----------

./loxodo.py
	runs Loxodo in GUI mode (if available)

./loxodo.py -h
	displays help about Loxodo's cmdline mode

./setup.py py2app
	creates a stand-alone .app (Mac OS X) 

./setup.py py2exe
	creates a stand-alone .exe (Windows)

./loxodo.py -i
	runs Loxodo in CLI Interactive mode


The Windows CE frontend of Loxodo is based on PocketPyGui by Alexandre Delattre

This directory contains SVN snapshot r46 (2008-09-07), 
retrieved from https://svn.nanl.de/svn/ppygui

Details on the terms of use and distribution of PocketPyGui can be found at the
top of each PocketPyGui source file.
